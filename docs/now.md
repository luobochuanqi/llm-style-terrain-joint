# 当前实现程度
1.  通用地貌文本生成：仅实现基础能力
    能通过自然语言生成雪山、沙漠、森林、峡谷等常见通用地貌，主流用Stable Diffusion系扩散模型，可输出高度图+纹理，但全是 先生成高度图再补纹理 的两阶段流水线，或两个独立分支分开生成。
2.  几何-纹理联合生成：仅有无文本的原型
    只有TerraFusion实现了 无文本/草图控制 的高度图+纹理隐空间联合生成；文本驱动的联合生成目前仅一篇2025年的预印本MESA，还是双分支分开生成，一致性极差。
3.  喀斯特/丹霞等特殊风格化地貌：无
    现有所有公开方案都只针对通用地貌，没有对喀斯特、丹霞、雅丹这类有专属地质特征的小众地貌做专项优化，生成结果的核心地质特征完全错误。
4.  文本控制精度：只能控大类，完全抓不住细节
    只能通过文本控制大的地貌类型（比如“雪山”“沙漠”），文本里的细粒度地质细节（比如“丹霞80度垂直陡崖”“喀斯特密集峰丛”）基本无法捕捉。
